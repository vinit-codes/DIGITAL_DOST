# Hype - Digital Agency Website

A modern, responsive digital agency website built with React, TypeScript, and Tailwind CSS. This project showcases various services, portfolio items, and includes a contact form.

## 🚀 Quick Start with GitHub Codespaces

This project is configured to work with GitHub Codespaces, providing a complete, cloud-based development environment.

### Using Codespaces

1. Click the "Code" button on the GitHub repository
2. Select "Open with Codespaces"
3. Click "New codespace"
4. Wait for the codespace to be created and initialized

The environment will be automatically configured with:

- Node.js
- npm
- Required VS Code extensions
- Project dependencies

### Running the Project in Codespaces

Once your codespace is ready:

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

The development server will start and you can view the site in the built-in browser.

## 🛠️ Local Development

If you prefer to develop locally:

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository

```bash
git clone https://github.com/vinit-codes/Hype.git
cd Hype
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```
